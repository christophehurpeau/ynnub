Styling text: inline

import 'ynnub/text/inline.scss';

<mark>: marked (highlighted) text

The mark elementrepresents a run of text in one document marked or highlighted for reference purposes, due to its relevance in another context.

MDN: mark element
Code
<p>
  This is an example of <mark>highlighted</mark> text.
</p>
Result

This is an example of highlighted text.

<del>: deleted text

The del elementrepresents a range of text that has been deleted from a document.

MDN: del element
Code
<p>
  This is an example of <del>deleted</del> text.
</p>
Result

This is an example of deleted text.

<s>: struck text

The s elementrepresents contents that are no longer accurate or no longer relevant and that therefore has been “struck” from the document.

MDN: s element
Code
<p>This is an example of <s>irreleveant</s> text.</p>
Result

This is an example of irreleveant text.

<ins>: inserted text

The ins elementrepresents a range of text that has been inserted (added) to a document.

MDN: ins element
Code

Result

This is an example of inserted text.

<u>: offset text conventionally styled with an underline

The u elementrepresents a span of text offset from its surrounding content without conveying any extra emphasis or importance.

MDN: u element

<u>: Chinese proper name marks

In Chinese writing, a proper name mark (Simplified Chinese: 专名号, zhuānmínghào; Traditional Chinese: 專名號) is an underline used to mark proper names, such as the names of people, places, dynasties, organizations. … This method of recognizing proper names in text is similar to the English use of a capital letter.

Proper name mark, Wikipedia

Wikipedia gives the following example, which can be formatted using <u>:

Code
<p>
  <u className="pnm">屈原</u>放逐，乃賦<cite className="bnm">離騒</cite>。<u className="pnm">左丘</u>失明，厥有<cite className="bnm">國語</cite>。（司馬遷 《<cite>報任安書</cite>》
</p>
Result

屈原放逐，乃賦離騒。左丘失明，厥有國語。（司馬遷 《報任安書》

<u>: Spell-checking feedback

import sTextSpellCheck 'ynnub/text/spell-check.scss';
Code
<p>This is an example of <u className={sTextSpellCheck.spellingError}>a spelling error</u>
{' '} and <u className={sTextSpellCheck.grammaticalError}>a grammatical error</u>.</p>
Result

This is an example of a spelling error and a grammatical error.

<i>: offset text conventionally styled with an italic

The i elementrepresents a span of text offset from its surrounding content without conveying any extra emphasis or importance, and for which the conventional typographic presentation is italic text; for example, a taxonomic designation, a technical term, an idiomatic phrase from another language, a thought, or a ship name.

MDN: i element
Code
<p>The Latin phrase <i>Veni, vidi, vici</i> is often mentioned in music, art, and literature.</p>
Result

The Latin phrase Veni, vidi, vici is often mentioned in music, art, and literature.

<b>: offset text conventionally styled in bold

The b elementrepresents a span of text offset from its surrounding content without conveying any extra emphasis or importance, and for which the conventional typographic presentation is bold text; for example, keywords in a document abstract, or product names in a review.
